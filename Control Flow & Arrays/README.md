# Grade Management System

A simple Java console-based Grade Management System developed using Object-Oriented Programming (OOP) concepts.

The application allows users to add student records, calculate grades, search students, generate reports, and analyze academic performance.

---

## Features

- Add new students
- Store marks for 5 subjects
- View all students
- Calculate average marks
- Assign grades automatically
- Search student by name
- Find top performer
- Generate performance report
- Input validation for marks (0–100)
- Menu-driven console application

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Arrays
- Exception Handling
- Scanner Class

## Grade Criteria

| Average Marks | Grade |
| ------------- | ----- |
| 90 - 100      | A+    |
| 80 - 89       | A     |
| 70 - 79       | B     |
| 60 - 69       | C     |
| 50 - 59       | D     |
| Below 50      | F     |

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Grade-Management-System.git
```

2. Open the project in your Java IDE (IntelliJ IDEA, Eclipse, VS Code, etc.)

3. Compile all Java files

```bash
javac *.java
```

4. Run

```bash
java GradeManagementSystem
```

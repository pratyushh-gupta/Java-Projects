# Student Information System

A complete command-line interface (CLI) application developed in Java to manage student records. It utilizes an in-memory database configuration allowing you to easily Add, View, Search, Update, and Delete student records with validation filters.

## 📁 Repository Structure

- **src/**: Contains the functional Java source code logic files.
- **docs/**: Detailed operational user documentation guides.
- **test_data/**: Mock baseline datasets for evaluating validation rules.

## 🚀 Quick Setup & Run Instructions

1. Open your system terminal/command prompt.
2. Navigate into the source folder directory:
   ```bash
   cd src
   ```
